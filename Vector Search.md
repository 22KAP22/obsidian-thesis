Vector Search is used in e-commerce, image retrieval, Retrieval Augmented Generation (RAG). k Nearest Neighbor Search is used to find the k nearest neighbors for a query q. kNN is expensive and precise doesn't need to be exact all the time, Approximated Nearest Neighbor (ANN) search is preferred. Indexes using clustering, tree, graph, quantization approaches speed up search significantly, but need extra time to build them in the first place. Some are agile and easily incorporate new vectors, where as others need to be build from scratch if too many new ones are inserted.
